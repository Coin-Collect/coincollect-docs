# Create a Wallet

To get started on CoinCollect, the first thing you'll need is to set up a wallet that supports Polygon Network (EVM based - ERC-20 ). Wallets are available both on desktop and mobiles. You'll need to choose the wallet that fits your needs best.&#x20;

{% hint style="info" %}

* ‚ùå **NEVER share your recovery phrases with anyone, under any circumstances.**
